# Smart Deadline Reminder System

## 📌 Overview

This project is a simple web-based application designed to help students manage academic deadlines efficiently. It allows users to add tasks, set deadlines, and receive reminders.

## 🚀 Features

* Add and manage tasks
* Set deadlines and priorities
* Get reminder alerts
* View upcoming deadlines

## 🛠️ Technologies Used

* python
* date time module
  
## How to Set Up
Make sure Python is installed on your computer
Download or clone this repository
Open the project folder

## Algorithm
Start the program.
Create an empty list to store tasks.
Display menu options (Add Task, Show Tasks, Check Reminders, Exit).
Take user choice.
If user selects:
Add Task → Enter task name and deadline, then store it.
Show Tasks → Display all saved tasks.
Check Reminders →
Get current date
Calculate days left for each task
Show reminders (today, tomorrow, or missed).
Exit → Stop the program.
Repeat steps 3–5 until user exits.
End program.

## How to Use
Choose option 1 to add a task
Enter task name and deadline (YYYY-MM-DD)
Choose option 2 to view tasks
Choose option 3 to check reminders
Choose option 4 to exit

## 📌 Example
Task: Assignment
Deadline: 2026-03-25

The system will remind you when the deadline is near or passed.
